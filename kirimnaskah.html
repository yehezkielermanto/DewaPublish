Menyiapkan file pengiriman

  1. Pastikan Anda sudah mempersiapkan semua file yang meliputi: (1) naskah, (2) sinopsis, (3) spesifikasi buku dan surat keaslian naskah, dan (4) data diri.
  2. Masukkan keempat file tersebut ke dalam 1 folder.
  3. Compress folder tersebut menjadi file .rar atau .zip.
  4. Naskah anda siap diunggah untuk dikirim.
